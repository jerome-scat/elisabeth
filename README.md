# Dashboard Contrôle de Gestion — Groupe Elisabeth 🌹

Dashboard analytique interactif pour le contrôle de gestion du groupe Elisabeth Luxembourg (organisation sociale fondée en 1672).

## 🔗 Live : [jerome-scat.github.io/elisabeth](https://jerome-scat.github.io/elisabeth)

## 📊 Fonctionnalités

- **Vue Synthèse** : KPIs consolidés, budget vs réel mensuel, répartition des charges, analyse des écarts budgétaires
- **Vue Analytique** : Indicateurs opérationnels, décomposition des charges, taux d'exécution mois par mois
- **Vue Consolidée** : Comparaison inter-entités, tableau de bord groupe complet
- **Fil de News** : Alertes financières, actualités par entité, filtres par tag

## 🛠️ Stack technique

- React 18 (CDN, no build step)
- Recharts pour les visualisations
- Tailwind CSS
- Déploiement GitHub Pages (fichier unique `index.html`)
